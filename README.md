# OmniAI Frontend

A modern, highly responsive, and premium React frontend for OmniAI, built with Vite, Tailwind CSS, Zustand, and Framer Motion.

## Prerequisites
- Node.js (v18+)
- npm or yarn

## Getting Started

1. **Install Dependencies**
   ```bash
   cd c:\Projects\OmniAI
   npm install
   ```

2. **Start the Development Server**
   ```bash
   npm run dev
   ```

3. **View the Application**
   Open the browser to `http://localhost:5173`.

## Design Highlights
- **Dark Mode Default:** Elegant shades of blue-gray (`#0F1117`, `#1E212B`) tailored for an AI workspace.
- **Smooth Animations:** Integrated with Framer Motion for sidebar toggles, message streaming, and dashboard stat entries.
- **State Management:** Fully optimized with Zustand for Chat and UI states.
- **High-End UI Components:** Custom scrollbars, layout transitions, animated bouncing loaders, and beautiful form inputs mimicking premium apps like ChatGPT and Notion.
