# OmniAI Frontend

A modern, highly responsive, and premium React frontend for OmniAI, built with Vite, Tailwind CSS, Zustand, and Framer Motion.

## Prerequisites
- Node.js (v18+)
- npm or yarn

## Getting Started

1. **Install Dependencies**
   ```bash
   cd c:\Projects\OmniAI
   npm install
   ```

2. **Start the Development Server**
   ```bash
   npm run dev
   ```

3. **View the Application**
   Open the browser to `http://localhost:5173`.

## Design Highlights
- **Dark Mode Default:** Elegant shades of blue-gray (`#0F1117`, `#1E212B`) tailored for an AI workspace.
- **Smooth Animations:** Integrated with Framer Motion for sidebar toggles, message streaming, and dashboard stat entries.
- **State Management:** Fully optimized with Zustand for Chat and UI states.
- **High-End UI Components:** Custom scrollbars, layout transitions, animated bouncing loaders, and beautiful form inputs mimicking premium apps like ChatGPT and Notion.

Designed with modern development practices in mind, this repository showcases a clean implementation optimized for scalability and readability.

## вљ™пёЏ Getting Started & Installation

### рџ“‹ Prerequisites
Ensure you have the runtime environment and managers installed for **Node.js/JavaScript/TypeScript, HTML/CSS/JavaScript**:
*   For JS/TS: **Node.js (v18+) & NPM**
*   For Python: **Python 3.10+ & pip**
*   For Flutter: **Flutter SDK**
*   For C#/.NET: **.NET SDK (v6.0+)**

### рџ’» Installation Walkthrough

1. Clone the repository to your local workspace:
   ```bash
   git clone https://github.com/karthikk20234119-cmd/omniai.git
   ```
2. Navigate into the project folder:
   ```bash
   cd OmniAI
   ```
3. Initialize the development environment and install dependencies:
   * **NodeJS**: `npm install`
   * **Python**: `pip install -r requirements.txt` (or activate your virtual environment first)
   * **Flutter**: `flutter pub get`
   * **.NET**: `dotnet restore`

4. Launch the application / script:
   * **NodeJS Dev Server**: `npm run dev`
   * **Python Core Script**: `python main.py` or `python app.py`
   * **FastAPI Server**: `uvicorn main:app --reload`
   * **Flutter Application**: `flutter run`
   * **.NET Core Solution**: `dotnet run`
